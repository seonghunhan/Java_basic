강의노트 4강_기본자료형

기본 자료형 : 데이터가 변수에 직접 저장
객체 자료형 : 객체 메모리 주소가 변수에 저장

기본 자료형
변수가 초기화되면 그 변수에 할당된 메모리에 데이터가 직접 들어간다
(각 자료에 따라 할당되는 크기가 다르다)

객체 자료형
객체(obj)를 메모리 어떤곳에 만들어 놓고 그 주소(obj의 주소)를 담고 있는 
레퍼런스(ox33ec45)를 이용한다

그래서 그 객체를 가르키고 있는 주소를 담고 있는 변수(i)를 사용!!!!
(C에서는 포인터, 자바에서는 레퍼런스)
레퍼런스는 기본적으로 4바이트 할당

정수형에서 보통 int를 쓰고 long은 너무 큼
byte는 256인데 -127~128에서 만약 150이면 오버플로우로 손실

char은 문자한개를 표현할때 사용 (그닥 많이 사용 X)

실수형 : double쓰면 모든 소수점 다 쓸 수 있을 정도로 많은크기 할당

논리형 : 0이 아닌수를 다 true라고 선언

문자열은 String이라는 자료형이고 엄밀히 말하면 객체 자료형에 속함


형 변환
작은크기가 큰크기에 데이터로 들어갈때는 데이터 누실 걱정 ㄴㄴ

큰크기의 데이터가 작은 크기의 데이터로 들어갈 때는 명시적 형 변환 사용
(실제로 큰크기의 데이터가 충분히 작다면 명시적 형 변환은 성립하는데
작은크기데이터보다 크기가 크다면 데이터 누실 발생)

데이터 누실 때문에 기본자료형에서는 기본적으로 작은->큰  자동(묵시적) 형 변환만
사용함

-----------6강_연산자

피연산자 개수에 의해 단항, 이항, 삼항으로 나뉜다
그닥 안중요

대입 연산자
'=' : 오른쪽 값을 왼쪽에 대입
'==' : 오른쪽과 왼쪽이 같다

복합 대입 연산자
x = x + 10
-> x += 10

증감 연산자
++x  :  x = x + 1 (전위 연산자)
x++  :  x를 출력후에 1을 더한것 (후위 연산자)
이거는 코드 붙여서 설명 ㄱㄱ
(화면에는 출력된 x는 10이지만 실제 메모리상 x는 11이다.)

논리 연산자
!x : x의 상태를 부정
( x가 false이면 !x는 true)
